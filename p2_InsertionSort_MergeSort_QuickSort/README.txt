In this file, there is a make file exist. It will make 3 exacutive files: consistentresultverification, sortedverification, and timealgorithms. 


To run each file individually, you do: 

g++ consistentresultverification.cxx -o consistentresultverification
./consistentresultverification ex1.json ex2.json

g++ sortedverification.cxx -o sortedverification
./sortedverification ex1.json

g++ timealgorithms.cxx -o timealgorithms
./timealgorithms ex1.json
